Aprendiendo Numeros primitivos

Los numeros primitivos no son un objeto y son inmutables veamos algunos de ellos:

Booleanos: True / False

Null: Sin Valor en lo absoluto

Undefine: Es una variable ya declararada que no se le asignado valor

Number: Integers, floats, etc

String: Cadena de Caracteres

Simbols: Valor Unico que no es igual a ningun otro valor


Palabras Reservadas

Aunque se nos olvide en muchas ocasiones, las palabras reservadas y la estructura correcta de los identificadores son algo importante, debemos conocer cuales son y evitar su uso, incluso en aquellos casos donde son aceptadas para algún tipo de artefacto del lenguaje. Aunque los editores modernos suelen avisar de que nos estamos equivocando, son muchas las ocasiones donde perdemos tiempo intentando comprender que está pasando porque hemos utilizado una palabra reservada sin darnos cuenta.

Arreglos

El objeto Array de JavaScript es un objeto global que es usado en la construcción de arrays, que son objetos tipo lista de alto nivel.

METODOS DE LOS ARREGLOS

Principales metodos en los arreglos

1. Largo: arr.LENGTH()

2. Acceder a un elemento del array mediante su posicion [0]

3. Conocer la ultima posicion .LENGTH -1

4. Recorrer un Arreglo mediante la funcion .FOREACH()

5. Añadir un elemento al final de un array con el metodo .PUSH()

6. Eliminar el ultimo elemento de un array .POP()

7. Añadir un elemento al principio de un Array .UNSHIFT

8. Elimina el primer elemento de un array .SHIFT()

9. Elimina un elemento mediante su posicion .SLICE()

10. Encontrar el indice de un elemento del array .INDEXOF()
